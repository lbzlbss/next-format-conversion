# Figma 插件 - 多媒体格式转换工具设计导入器

这个 Figma 插件可以将"多媒体格式转换工具"的完整 UI 设计自动导入到 Figma 中。

## 功能特性

✅ **完整的页面布局** - 1200x1400px 的完整设计框架  
✅ **渐变背景** - 紫色渐变背景和装饰元素  
✅ **玻璃态效果** - 现代化的玻璃模态设计风格  
✅ **7个功能标签页** - GIF转WebP、MP4压缩等所有功能  
✅ **交互组件** - 按钮、输入框、进度条、卡片等  
✅ **设计规范** - 完整的颜色、间距、圆角等设计系统  

## 安装步骤

### 1. 准备插件文件

确保你有以下三个文件：
- `manifest.json` - 插件配置文件
- `code.js` - 插件主逻辑代码
- `ui.html` - 插件用户界面

### 2. 在 Figma 中安装插件

#### 方法一：通过 Figma 桌面应用（推荐）

1. 打开 Figma 桌面应用
2. 点击菜单栏 **Plugins** → **Development** → **Import plugin from manifest...**
3. 选择 `manifest.json` 文件所在的文件夹
4. 插件将被添加到你的开发插件列表中

#### 方法二：通过 Figma 网页版

1. 打开 Figma 网页版
2. 在任意文件中，点击菜单栏 **Plugins** → **Development** → **Import plugin from manifest...**
3. 选择 `manifest.json` 文件
4. 插件将被添加到你的开发插件列表中

### 3. 运行插件

1. 在 Figma 中打开你想要导入设计的文件（或创建新文件）
2. 点击菜单栏 **Plugins** → **Development** → **Next Format Conversion Design Importer**
3. 在弹出的插件窗口中，点击 **"开始导入设计"** 按钮
4. 等待几秒钟，设计将自动创建在当前页面中

## 导入的内容

插件将创建以下设计元素：

### 页面结构
- **主容器** (1200x1400px)
  - 渐变背景 (#667eea → #764ba2 → #f093fb)
  - 背景装饰圆形（带模糊效果）
  - 玻璃态主内容区域

### 组件
- **标题** - 渐变文字效果
- **标签页导航** - 7个功能标签
  - GIF 转 WebP
  - MP4 压缩
  - GIF 转 MP4
  - MP4 获取首帧
  - 图片压缩
  - GIF 压缩
  - AI 图像生成

### 功能区域
- **上传区域** - 文件上传卡片和按钮
- **文件列表** - 显示已上传文件的状态
- **配置面板** - 压缩参数配置（滑块控件）
- **操作按钮** - 批量压缩和下载按钮

## 使用提示

1. **字体要求** - 插件使用 Inter 字体，确保你的 Figma 中已安装 Inter 字体
2. **编辑设计** - 导入后，所有元素都是可编辑的 Figma 图层
3. **组件化** - 你可以将重复的元素转换为 Figma 组件以便复用
4. **导出** - 可以直接从 Figma 导出为图片或代码

## 故障排除

### 插件无法运行
- 确保使用的是 Figma 桌面应用（网页版可能有限制）
- 检查 `manifest.json` 文件格式是否正确
- 确保所有三个文件在同一目录下

### 字体显示问题
- 在 Figma 中安装 Inter 字体
- 或者在代码中修改为你已有的字体

### 设计元素位置不对
- 确保在空白页面或有足够空间的位置运行插件
- 插件会自动将设计放置在 (0, 0) 位置

## 自定义修改

如果你想修改设计，可以编辑 `code.js` 文件中的 `DESIGN_TOKENS` 对象：

```javascript
const DESIGN_TOKENS = {
  colors: {
    primary: { r: 0.4, g: 0.494, b: 0.918 }, // 修改主色
    // ... 其他颜色
  },
  spacing: {
    md: 16, // 修改间距
    // ... 其他间距
  },
  // ... 其他设计规范
};
```
