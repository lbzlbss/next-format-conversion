# 快速开始指南 - 5分钟导入设计到 Figma

## 🚀 三步完成导入

### 第一步：安装插件（2分钟）

1. **打开 Figma 桌面应用**
   
2. **导入插件**
   - 点击顶部菜单：`Plugins` → `Development` → `Import plugin from manifest...`
   - 浏览到 `next-format-conversion/figma-plugin` 文件夹
   - 选择 `manifest.json` 文件
   - 点击"打开"

3. **确认安装**
   - 插件会出现在 `Plugins` → `Development` 菜单下
   - 名称：**Next Format Conversion Design Importer**

### 第二步：运行插件（1分钟）

1. **打开或创建 Figma 文件**
   - 可以是新文件或现有文件
   - 建议使用空白页面以便查看效果

2. **启动插件**
   - 点击菜单：`Plugins` → `Development` → `Next Format Conversion Design Importer`
   - 或使用快捷键：`Cmd/Ctrl + /` 搜索插件名称

3. **开始导入**
   - 在弹出窗口中点击 **"开始导入设计"** 按钮
   - 等待 3-5 秒完成导入

### 第三步：查看结果（2分钟）

✅ 导入完成后，你会看到：
- 完整的页面设计（1200x1400px）
- 紫色渐变背景
- 玻璃态卡片效果
- 7个功能标签页
- 所有交互组件

## 🎨 接下来做什么？

### 1. 编辑设计
- 所有元素都是可编辑的 Figma 图层
- 双击文本进行修改
- 调整颜色、大小、位置等

### 2. 创建组件
- 选中重复元素（如按钮、卡片）
- 右键 → `Create Component`
- 便于在其他页面复用

### 3. 导出资源
- 选中元素 → 右侧面板 → `Export`
- 支持 PNG、JPG、SVG、PDF 等格式
- 可设置不同分辨率（1x, 2x, 3x）

### 4. 生成代码
- 选中元素 → 右侧面板 → `Code`
- 查看 CSS、iOS、Android 代码
- 复制到你的项目中

## ⚡ 常见问题

### Q: 字体显示不正确？
**A:** 确保已安装 Inter 字体
- 下载：https://fonts.google.com/specimen/Inter
- 或在 Figma 中搜索并启用 Inter 字体

### Q: 插件找不到？
**A:** 检查安装路径
- 确保三个文件在同一文件夹：
  - `manifest.json`
  - `code.js`
  - `ui.html`

### Q: 导入后位置不对？
**A:** 设计会自动放在 (0, 0) 位置
- 使用 `Cmd/Ctrl + 2` 缩放到适合视图
- 或手动拖动到合适位置

### Q: 想修改颜色或样式？
**A:** 两种方式：
1. **在 Figma 中直接修改**（推荐）
   - 选中元素 → 修改属性
2. **修改插件代码**
   - 编辑 `code.js` 中的 `DESIGN_TOKENS`
   - 重新运行插件
