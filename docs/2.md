# 视频去水印功能实现方案

本文档为 **Video Watermark Remover** 功能的完整实现规范，供 Cursor AI 助手理解并生成代码。

---

## 1. 功能概述

为项目添加 **视频去水印** 功能，支持：
- 交互式框选水印区域
- 基于 FFmpeg 的轻量级去水印（`delogo` 滤镜）
- 可选的 AI 增强去水印（ProPainter）
- 导出处理后的视频

## 4. 实现方案

### 4.1 方案选择

#### 方案 A：轻量级方案（推荐用于 Vercel）
- **技术**：FFmpeg `delogo` 滤镜
- **优点**：计算快、无需额外依赖、适合 Serverless
- **缺点**：效果一般，可能有轻微模糊
- **适用场景**：背景较单一、对质量要求不高的场景

#### 方案 B：专业级方案（推荐用于生产）
- **技术**：FFmpeg + ProPainter（Python）
- **优点**：效果好，保留纹理细节
- **缺点**：需要独立服务（Docker + GPU）
- **适用场景**：高质量要求、商业用途

#### 方案 C：混合方案（推荐）
- **策略**：
  - 小文件（< 10MB）：使用方案 A（直接处理）
  - 大文件（≥ 10MB）：提交到任务队列，使用方案 B
- **优点**：兼顾性能与质量

'use client';

import { useState, useRef, useEffect, useCallback } from 'react';
import { Upload, Button, Card, InputNumber, Switch, Slider, message } from 'antd';
import { InboxOutlined, PlayCircleOutlined, PauseCircleOutlined, DownloadOutlined } from '@ant-design/icons';

const VideoWatermarkRemover = () => {
  const [videoUrl, setVideoUrl] = useState('');
  const [videoFile, setVideoFile] = useState(null);
  const [mask, setMask] = useState({ x: 0, y: 0, width: 100, height: 50 });
  const [useAI, setUseAI] = useState(false);
  const [smooth, setSmooth] = useState(false);
  const [keepAudio, setKeepAudio] = useState(true);
  const [isProcessing, setIsProcessing] = useState(false);
  const [progress, setProgress] = useState(0);
  const videoRef = useRef(null);
  const canvasRef = useRef(null);
  const [isDragging, setIsDragging] = useState(false);
  const [dragStart, setDragStart] = useState(null);

  // 上传处理
  const handleUpload = (file) => {
    const url = URL.createObjectURL(file);
    setVideoUrl(url);
    setVideoFile(file);
    
    // 初始化水印区域为右下角（常见位置）
    const video = videoRef.current;
    video.addEventListener('loadedmetadata', () => {
      setMask({
        x: video.videoWidth - 150,
        y: video.videoHeight - 60,
        width: 140,
        height: 50
      });
    });
    
    return false;
  };

  // Canvas 拖拽处理
  const handleCanvasMouseDown = (e) => {
    const canvas = canvasRef.current;
    const rect = canvas.getBoundingClientRect();
    const x = e.clientX - rect.left;
    const y = e.clientY - rect.top;
    setDragStart({ x, y });
    setIsDragging(true);
  };

  const handleCanvasMouseMove = (e) => {
    if (!isDragging || !dragStart) return;
    const canvas = canvasRef.current;
    const rect = canvas.getBoundingClientRect();
    const x = e.clientX - rect.left;
    const y = e.clientY - rect.top;
    
    setMask({
      x: Math.min(dragStart.x, x),
      y: Math.min(dragStart.y, y),
      width: Math.abs(x - dragStart.x),
      height: Math.abs(y - dragStart.y)
    });
  };

  const handleCanvasMouseUp = () => {
    setIsDragging(false);
    setDragStart(null);
  };

  // 开始处理
  const handleProcess = async () => {
    if (!videoFile) {
      message.warning('请先上传视频');
      return;
    }

    setIsProcessing(true);
    setProgress(0);

    try {
      const formData = new FormData();
      formData.append('file', videoFile);
      formData.append('maskX', Math.round(mask.x));
      formData.append('maskY', Math.round(mask.y));
      formData.append('maskWidth', Math.round(mask.width));
      formData.append('maskHeight', Math.round(mask.height));
      formData.append('useAI', useAI);
      formData.append('smooth', smooth);
      formData.append('keepAudio', keepAudio);

        method: 'POST',
        body: formData
      });

      if (!res.ok) {
        const err = await res.json();
        throw new Error(err.error || '处理失败');

      const blob = await res.blob();
      const url = URL.createObjectURL(blob);
      setVideoUrl(url);
      
      const a = document.createElement('a');
      a.href = url;
      a.download = `no-watermark_${videoFile.name}`;
      a.click();
      
      message.success('处理成功');
    } catch (error) {
      message.error(error.message || '处理失败');
    } finally {
      setIsProcessing(false);
      setProgress(0);
    }
  };

  return (
    <div className="space-y-4">
      {/* 上传区 */}
      <Card title="上传视频">

          accept=".mp4"
          showUploadList={false}
          beforeUpload={handleUpload}
        >
          <p className="ant-upload-drag-icon">
            <InboxOutlined />
          </p>
          <p className="ant-upload-text">点击或拖拽 MP4 文件到此区域</p>
        </Upload.Dragger>
      </Card>

      {/* 预览区 */}
      {videoUrl && (
        <Card title="视频预览">
          <div className="relative">
            <video
              ref={videoRef}
              src={videoUrl}
              controls
              className="w-full rounded-lg"
              onPlay={() => setVideoUrl(videoUrl)}
            />
            <canvas
              ref={canvasRef}
              className="absolute top-0 left-0 pointer-events-none"
              style={{ width: '100%', height: '100%' }}
            />
          </div>
          
          {/* 水印区域设置 */}
          <div className="mt-4 grid grid-cols-4 gap-2">
            <div>
              <label className="text-xs font-semibold">X 坐标</label>
              <InputNumber
                className="w-full"
                value={mask.x}
                onChange={(v) => setMask({ ...mask, x: v })}
                min={0}
              />
            </div>
            <div>
              <label className="text-xs font-semibold">Y 坐标</label>
              <InputNumber
                className="w-full"
                value={mask.y}
                onChange={(v) => setMask({ ...mask, y: v })}
                min={0}
              />
            </div>
            <div>
              <label className="text-xs font-semibold">宽度</label>
              <InputNumber
                className="w-full"
                value={mask.width}
                onChange={(v) => setMask({ ...mask, width: v })}

            <div>
              <label className="text-xs font-semibold">高度</label>

                className="w-full"

### 5.2 后端 API

#### 文件：`src/app/api/remove-watermark/route.js`

```javascript
import { NextResponse } from 'next/server';
import fs from 'fs';
import path from 'path';
import os from 'os';
import ffmpeg from 'fluent-ffmpeg';

// 配置 ffmpeg 路径
let ffmpegPath;
if (process.env.NODE_ENV === 'production') {
  ffmpegPath = '/usr/bin/ffmpeg';
} else {
  ffmpegPath = path.join(process.cwd(), 'public', 'ffmpeg', 'ffmpeg');
}

if (fs.existsSync(ffmpegPath)) {
  ffmpeg.setFfmpegPath(ffmpegPath);
} else {
  console.error('未找到 ffmpeg');
}

const tempDir = os.tmpdir();

export async function POST(request) {
  try {
    const formData = await request.formData();
    const file = formData.get('file');
    const maskX = parseInt(formData.get('maskX'));
    const maskY = parseInt(formData.get('maskY'));
    const maskWidth = parseInt(formData.get('maskWidth'));
    const maskHeight = parseInt(formData.get('maskHeight'));
    const useAI = formData.get('useAI') === 'true';
    const smooth = formData.get('smooth') === 'true';
    const keepAudio = formData.get('keepAudio') !== 'false';

    // 验证文件
    if (!file || !file.type || !file.type.startsWith('video/mp4')) {
      return NextResponse.json({ error: '只支持 MP4 格式' }, { status: 400 });
    }

    // 验证参数
    if (isNaN(maskX) || isNaN(maskY) || isNaN(maskWidth) || isNaN(maskHeight)) {
      return NextResponse.json({ error: '水印区域参数无效' }, { status: 400 });
    }

    // 创建临时文件
    const tempInput = path.join(tempDir, `input_${Date.now()}.mp4`);
    const tempOutput = path.join(tempDir, `output_${Date.now()}.mp4`);

    try {
      // 写入临时文件
      const buffer = await file.arrayBuffer();
      fs.writeFileSync(tempInput, Buffer.from(buffer));

      // 构建 ffmpeg 命令
      let command = ffmpeg(tempInput);

      // 添加 delogo 滤镜
      const delogoFilter = `delogo=x=${maskX}:y=${maskY}:w=${maskWidth}:h=${maskHeight}`;
      const filterArgs = smooth ? `${delogoFilter}:show=0` : delogoFilter;
      command = command.videoFilters(filterArgs);

      // 音频处理
      if (keepAudio) {
        command = command.audioFilters('anull'); // 保留音频
      } else {
        command = command.noAudio();
      }

      // 输出设置
      command = command.outputOptions([
        '-c:v libx264',
        '-crf 23',
        '-preset medium',
        '-c:a aac',
        '-b:a 128k',
        '-movflags +faststart'
      ]);

      // 超时保护（Vercel Pro 限制为 60 秒）
      const timeout = 60000;

      // 执行处理
      await new Promise((resolve, reject) => {
        const timeoutId = setTimeout(() => {
          reject(new Error('处理超时'));
        }, timeout);

        command
          .on('end', () => {
            clearTimeout(timeoutId);
            resolve();
          })
          .on('error', (err) => {
            clearTimeout(timeoutId);
            reject(err);
          })
          .on('progress', (progress) => {
            if (progress.percent) {
              console.log(`处理进度: ${progress.percent}%`);
            }
          })
          .save(tempOutput);
      });

      // 读取输出文件
      const outputBuffer = fs.readFileSync(tempOutput);
      const outputSize = outputBuffer.length;

      // 清理临时文件
      fs.unlinkSync(tempInput);
      fs.unlinkSync(tempOutput);

      // 返回结果
      return new Response(outputBuffer, {
        headers: {
          'Content-Type': 'video/mp4',
          'Content-Disposition': `attachment; filename="no-watermark_${file.name}"`,
          'Content-Length': outputSize.toString()
        }
      });

    } catch (error) {
      // 清理临时文件
      try {
        if (fs.existsSync(tempInput)) fs.unlinkSync(tempInput);
        if (fs.existsSync(tempOutput)) fs.unlinkSync(tempOutput);
      } catch (_) {}

      console.error('处理失败:', error);
      return NextResponse.json({ error: error.message || '处理失败' }, { status: 500 });
    }

  } catch (error) {
    return NextResponse.json({ error: error.message || '请求失败' }, { status: 500 });
  }
}
```

## 7. 扩展方案（ProPainter）

### 7.1 Python 处理脚本

#### 文件：`scripts/remove_watermark.py`

```python
import cv2
import numpy as np
import torch
from proPainter import ProPainter

def remove_watermark(video_path, mask_coords, output_path, use_ai=True, smooth=True):
    """
    移除视频水印
    
    Args:
        video_path: 输入视频路径
        mask_coords: 水印区域 (x, y, w, h)
        output_path: 输出视频路径
        use_ai: 是否使用 AI 增强
        smooth: 是否平滑处理
    """
    # 读取视频
    cap = cv2.VideoCapture(video_path)
    fps = cap.get(cv2.CAP_PROP_FPS)
    width = int(cap.get(cv2.CAP_PROP_FRAME_WIDTH))
    height = int(cap.get(cv2.CAP_PROP_FRAME_HEIGHT))
    
    # 创建视频写入器
    fourcc = cv2.VideoWriter_fourcc(*'mp4v')
    out = cv2.VideoWriter(output_path, fourcc, fps, (width, height))
    
    # 初始化 ProPainter
    if use_ai:
        model = ProPainter(device='cuda' if torch.cuda.is_available() else 'cpu')
    
    x, y, w, h = mask_coords
    
    while True:
        ret, frame = cap.read()
        if not ret:
            break
        
        # 复制帧
        result = frame.copy()
        
        if use_ai:
            # AI 去水印
            mask = np.zeros((height, width), dtype=np.uint8)
            mask[y:y+h, x:x+w] = 255
            result = model.inpaint(frame, mask)
        else:
            # FFmpeg delogo 算法（简化版）
            roi = frame[y:y+h, x:x+w].copy()
            blurred = cv2.medianBlur(roi, 15)
            result[y:y+h, x:x+w] = blurred
        
        # 平滑处理
        if smooth and not use_ai:
            result = cv2.bilateralFilter(result, 9, 75, 75)
        
        out.write(result)
    
    cap.release()
    out.release()
```

## 10. 常见问题

### Q1：Vercel Serverless 能处理视频吗？
**A**：可以，但有限制：
- 小文件（< 10MB）：直接处理

### Q2：FFmpeg delogo 效果不好怎么办？
**A**：建议使用 ProPainter（需要独立服务）

### Q3：如何减少处理时间？
**A**：
- 降低视频分辨率
- 减少帧率
- 使用硬件加速（需要 GPU 服务器）
